Introduction

Supercar attache une grande importance à la protection de votre vie privée et s'engage à sécuriser vos données personnelles. Cette politique de confidentialité a pour objectif de vous informer sur le traitement de vos données personnelles lors de votre visite sur nos sites web, quel que soit votre lieu de connexion. Elle vous éclaire également sur vos droits en matière de confidentialité et sur les protections légales en place.

Cette politique de confidentialité est présentée sous forme de couches pour que vous puissiez accéder aux domaines spécifiques mentionnés ci-dessous.

Objectif de cette politique de confidentialité :

Cette politique de confidentialité a pour but de vous fournir des informations détaillées sur la collecte et le traitement de vos données personnelles par Supercar. Cela inclut les données que vous pourriez nous fournir via notre site web dans différentes situations telles que :
  • Arrivez sur notre site web depuis un site de médias sociaux tel que Facebook, Instagram ou LinkedIn ;
  • Remplissez un formulaire de rappel/demande de renseignements ;
  • Vous inscrivez sur notre site pour postuler à un cours ou programme, à une formation d'entreprise/exécutive, à une formation professionnelle (en ligne, en apprentissage à distance ou sur le campus), que ce soit pour vous-même ou pour des employés ou d'autres membres de votre organisation qui pourraient être ou devenir nos clients ;
  • Vous abonnez à des documents marketing ;
  • Vous nous fournissez vos données personnelles par tout autre moyen.

Il est important de noter que ce site web n'est pas destiné aux enfants, et nous ne collectons délibérément aucune donnée concernant les enfants sur cette plateforme. Cependant, si Supercar traite des données personnelles d'enfants dans le cadre de ses activités commerciales, nous nous assurons d'obtenir les garanties et les consentements appropriés des parents ou des responsables, le cas échéant.

Il est recommandé de lire attentivement cette politique de confidentialité en parallèle avec d'autres avis de confidentialité spécifiques que nous pourrions fournir lors de certaines occasions où nous collectons ou traitons vos données personnelles. Ceci afin que vous ayez une compréhension complète de la manière dont nous utilisons vos données.

RGPD

Finalité du traitement des données:

Les données personnelles que vous nous fournissez sont utilisées dans le but de gérer les contacts, les demandes d'essai, les inscriptions sur notre site web, ainsi que pour l'authentification des utilisateurs.

Base juridique du traitement :

Nous traitons vos données personnelles sur la base de votre consentement, de l'exécution d'un contrat ou de notre intérêt légitime à fournir nos services et à gérer notre entreprise.

Droits des utilisateurs :
  • Droit d'accès :
    • avez le droit de demander des informations sur les données personnelles que nous détenons à votre sujet
    • Vous pouvez demander à obtenir une copie de ces données, ainsi que des informations sur la manière dont elles sont traitées.
  • Droit de rectification :
    • Si vous constatez que les données personnelles que nous détenons à votre sujet sont inexactes ou incomplètes, vous avez le droit de demander leur correction ou leur mise à jour.
  • Droit de suppression:
    • Vous avez le droit de demander la suppression de vos données personnelles si elles ne sont plus nécessaires aux fins pour lesquelles elles ont été collectées, si vous retirez votre consentement au traitement ou si leur traitement est illicite.
  • Droit de restriction du traitement:
    • Vous pouvez demander la limitation du traitement de vos données personnelles dans certaines circonstances, par exemple si vous contestez l'exactitude des données ou si leur traitement est illicite, mais que vous ne souhaitez pas qu'elles soient supprimées.
  • Droit à la portabilité des données :
    • Dans certaines situations, vous avez le droit de recevoir vos données personnelles dans un format structuré, couramment utilisé et lisible par machine, et de les transmettre à un autre responsable du traitement.
  • Droit d'opposition au traitement :
    • Vous avez le droit de vous opposer au traitement de vos données personnelles dans certaines circonstances, notamment si le traitement est effectué à des fins de marketing direct ou si vous contestez l'intérêt légitime sur lequel nous nous appuyons pour traiter vos données.
  • Droit à l'oubli :
    • Également connu sous le nom de droit à l'effacement, ce droit vous permet de demander la suppression de vos données personnelles lorsque leur traitement n'est plus nécessaire aux fins pour lesquelles elles ont été collectées, ou si vous avez retiré votre consentement au traitement.

Conservation des données :
  • Pour les données liées aux essais de véhicules, nous conservons vos données personnelles pendant 1 mois après l'essai, sauf si vous nous donnez votre consentement pour les conserver plus longtemps afin de vous informer des nouveautés.
  • Pour les données de contact, nous les conservons pendant la durée des échanges avec le potentiel client, et les supprimons ensuite sauf si vous nous donnez votre consentement pour les conserver plus longtemps.
